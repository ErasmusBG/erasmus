# erasmus

Connecting schools, VET centres, and municipalities with vetted European hosts, premium course providers, and end-to-end mobility logistics—delivered with institutional precision and full Erasmus+ compliance.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
